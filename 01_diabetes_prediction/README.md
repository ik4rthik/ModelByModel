## 🚀 About The Project

An interactive Machine Learning web app that predicts diabetes using multiple classification models while demonstrating real-world ML concepts like preprocessing, feature scaling, model comparison, and hyperparameter tuning.

---

## 🤖 Models Used

* Logistic Regression
* SVM
* Random Forest
* Gradient Boosting

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* Streamlit

---

## 📚 What I Learned

* Building complete ML pipelines
* Feature scaling and preprocessing
* Comparing ML model performance
* Hyperparameter tuning with GridSearchCV
* Evaluating models using ROC-AUC, F1-score & confusion matrix
* Deploying ML applications using Streamlit
